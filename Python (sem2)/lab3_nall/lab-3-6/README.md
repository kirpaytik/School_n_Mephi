# Лабораторная 3. Вариант 6. 
### Задача 1. Положительные соседи
Напишите функцию `plus_sign(str) -> bool`, которая принимает строку и возвращает True, если каждая буква в этой строке окружена знаками плюс.  
В противном случае функция должна вернуть False.
###  Входные данные

Строка 
### Выходные данные

True или False
#### Пример 1

**Входные данные**

```
plus_sign("+f+d+c+#+f+")
```

Выходные данные

```
True
```
#### Пример 2

**Входные данные**

```
plus_sign("+d+=3=+s+")
```

Выходные данные

```
True
```
#### Пример 3

**Входные данные**

```
plus_sign("f++d+g+8+") 
```

Выходные данные

```
False
```
#### Пример 4

**Входные данные**

```
plus_sign("+s+7+fg+r+8+") 
```

Выходные данные

```
False
```
